# Manufacturing Quality Control with Control Charts

## 🏭 Project Overview
This project analyzes manufacturing process data using control charts to monitor quality control and detect process variations. The analysis helps identify bottlenecks and improve production line efficiency.

## 🎯 Objectives
- Implement statistical process control using control charts
- Monitor manufacturing quality and detect anomalies
- Identify process bottlenecks and inefficiencies
- Provide recommendations for quality improvement
- Optimize production line performance

## 📊 Methodology
- Statistical process control (SPC) implementation
- Control chart analysis for quality monitoring
- Process capability analysis and improvement
- Root cause analysis for quality issues
- Performance optimization recommendations

## 🛠️ Tools Used
- Python (Jupyter Notebook)
- Pandas for manufacturing data analysis
- Statistical libraries for control chart calculations
- Matplotlib/Seaborn for quality control visualizations
- NumPy for statistical computations

- `Samatrix_6th_Project_Manufacturing.ipynb` - Complete quality control analysis and control charts
