# Portfolio Risk Assessment & 1-Day 95% VaR Calculation

## 📊 Project Overview
This project focuses on financial risk assessment using Value at Risk (VaR) methodology. The analysis calculates the 1-day 95% VaR for a portfolio to estimate potential losses under normal market conditions.

## 🎯 Objectives
- Conduct comprehensive risk analysis on financial portfolio data
- Calculate 1-day 95% Value at Risk using statistical methods
- Provide insights for risk management and investment decision-making

## 📈 Methodology
- Statistical analysis of historical financial data
- VaR calculation using parametric and historical simulation approaches
- Risk assessment and portfolio optimization insights

## 🛠️ Tools Used
- Python (Jupyter Notebook)
- Pandas, NumPy for data manipulation
- Statistical libraries for VaR calculations
- Matplotlib/Seaborn for visualization

## 📁 Files
- `Samatrix_1st_Project_VAR.ipynb` - Complete analysis and VaR calculations
